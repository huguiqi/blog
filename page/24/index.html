docker 换国内镜像

docker 换国内镜像

搞了一天，终于把docker的地址换成国内镜像了，搞死我了。。。。。。

把链接附上，这是国内号称永久免费的镜像服务，需要注册
https://dashboard.daocloud.io/mirror

安装或升级Mac OS版本Boot2Docker

Docker 1.3.2 版本以上支持加速器，如果您没有安装 Docker 或者版本较旧，请安装或升级。

参考 Boot2Docker 安装文档。高速下载 Boot2Docker

配置Docker加速器

请确认你的 Boot2Docker 已经启动，并执行

boot2docker up

boot2docker ssh “echo $’EXTRA_ARGS=\”–registry-mirror=http://xxxx.m.daocloud.io\“‘ | sudo tee -a /var/lib/boot2docker/profile”

boot2docker restart

其中xxxx是你登陆以后分配给你的mirror

将 Mac 中的文件夹共享到虚拟机

将 Mac 中的文件夹共享到虚拟机

我用的是vmware Fusion 7.0

要在 Linux 或 Solaris 虚拟机中配置共享文件夹，请执行以下操作：

注意：FreeBSD 不支持共享文件夹。

启动 VMware Fusion。
关闭虚拟机。
单击虚拟机 > 设置。
单击共享。
在 Fusion 8.x 和 7.x 中，选择启用共享文件夹选项。在 Fusion 6.x 、5.x 和 4.x 中，将共享文件夹选项选为开启。在 Fusion 3.x 中，选择在 Mac 上共享文件夹选项。
